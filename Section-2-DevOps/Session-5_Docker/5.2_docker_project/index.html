Apache HTTP Server running in Docker!

Base Image: httpd:2.4

Port: 80

Document Root: /usr/local/apache2/htdocs/

How to run this:

# Build the image
docker build -f Dockerfile.httpd -t my-httpd .

# Run the container
docker run -d -p 8080:80 --name my-apache my-httpd

# Access at: http://localhost:8080
